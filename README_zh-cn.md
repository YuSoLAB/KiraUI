# KiraUI

一款开源的博客工具与内容发布平台，具有轻量设计、简单易用、易于二次开发的特点。

## 下载与使用

直接从本页面下载压缩包，上传至服务器根目录，解压即可使用。首次部署时，访问`yourdomain.com/admin/admin`完成初始配置。

## 推荐环境

可访问`yourdomain.com/test.php`检测配置。强烈推荐使用 Apache 2.4 + PHP 8.4 + MySQL 9.1.0 环境。若使用 Nginx，需进行额外配置。
