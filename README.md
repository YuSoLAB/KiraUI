# KiraUI
An open-source specialized blog tool and content publishing platform, featuring lightweight design, ease of use, and easy customization.

## Download and Use

Download the zip package directly from this page, upload it to the root directory of your server, and extract it to use. For initial deployment, visit `yourdomain.com/admin/admin` to complete the initial configuration.
## Recommended Environment
You can visit `yourdomain.com/test.php` to check the configuration. Apache 2.4 + PHP 8.4 + MySQL 9.1.0 is highly recommended. Additional configuration is required if using Nginx.

## Additional Notes
The project is currently in the early stages of development. Many tasks have yet to be started, many features are not yet implemented, and many bugs remain unresolved. We are also looking for interested programmers to assist with development. If you are interested, please feel free to reach out to us.
